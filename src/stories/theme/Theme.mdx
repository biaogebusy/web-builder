# 介绍

<img src="/assets/storybook/assets/builder.png" />

信使默认内置以下几种主题：

- 浅色主题（默认）
- 暗黑主题
- 粉色主题
- 蓝色主题
- 绿色主题

在主题配置文件 `src/theme/theme.scss` 中可查看并根据项目情况删除或者添加主题，我们推荐严格按照此方式来添加主题，即不会在具体的组件中写相关的颜色信息，而是在 theme 中根据变量严格设置，才能够保证组件统一颜色。

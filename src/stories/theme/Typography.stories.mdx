### Material 字体排版配置及应用

| Class 类名                      | 对应变量名    | 应用 HTML 元素 |
| ------------------------------- | ------------- | -------------- |
| .mat-display-4                  | $display-4    | Hero 标题      |
| .mat-display-3                  | $display-3    | Hero 标题      |
| .mat-display-2                  | $display-2    | Hero 标题      |
| .mat-display-1                  | $display-1    | Hero 标题      |
| .mat-h1 or .mat-headline        | $headline     | `<h1>`         |
| .mat-h2 or .mat-title           | $title        | `<h2>`         |
| .mat-h3 or .mat-subheading-2    | $subheading-2 | `<h3>`         |
| .mat-h4 or .mat-subheading-1    | $subheading-1 | `<h4>`         |
| .mat-h5                         | none          | `<h5>`         |
| .mat-h6                         | none          | `<h6>`         |
| .mat-body or .mat-body-1        | body-1        | body text      |
| .mat-body-strong or .mat-body-2 | body-2        | none           |
| .mat-small or .mat-caption      | caption       | none           |

自定义字体配置生成

```ts
$custom-typography: mat-typography-config(
  $font-family: $font-family,
  $display-4: mat-typography-level(48px, 1.618, 300, $font-family: $font-family),
  $display-3: mat-typography-level(40px, 1.618, 400, $font-family: $font-family),
  $display-2: mat-typography-level(32px, 1.618, 400, $font-family: $font-family),
  $display-1: mat-typography-level(28px, 1.618, 400, $font-family: $font-family),
  $headline: mat-typography-level(24px, 1.618, 400, $font-family: $font-family),
  $title: mat-typography-level(20px, 1.618, 500, $font-family: $font-family),
  $body-1: mat-typography-level(16px, 1.618, 400),
);
```

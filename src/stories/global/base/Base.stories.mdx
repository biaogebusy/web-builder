## 组件动画

有些组件设置有 gsap 动画，可全局开启或者禁用。

```json
{
  // ...
  "animate": true
}
```

## 加载状态的配置

- 全局页面的 load 效果
- api 请求时顶部 loadingBar 的效果

```json
{
  //..
  "loading": true,
  "loadingBar": true
}
```

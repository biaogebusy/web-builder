## 编辑器配置

相关的配置可参考官网[quilljs](https://quilljs.com/)，另外组件的配置优于全局配置。

```json
{
  //..
  "editor": {
    "modules": {
      "toolbar": [
        [
          {
            "header": [1, 2, 3, 4, 5, 6, false]
          }
        ],
        ["bold", "italic", "underline", "strike"],
        ["clean"],
        [
          {
            "background": []
          },
          {
            "color": []
          }
        ],
        [
          {
            "align": []
          }
        ],
        [
          {
            "list": "ordered"
          },
          {
            "list": "bullet"
          }
        ],
        [
          {
            "indent": "-1"
          },
          {
            "indent": "+1"
          }
        ],
        ["link"],
        ["blockquote", "code-block"]
      ]
    }
  }
}
```

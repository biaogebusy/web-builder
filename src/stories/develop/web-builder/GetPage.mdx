## 获取某个页面

```json
POST {{apiUrl}}/api/v3/landingPage/json/{nid}
Content-type: application/json
withCredentials: true
X-CSRF-Token: {{csrfToken}}
```

## 支持多语言

- 默认语言：http://example.com/api/v3/landingPage/json/54
- 其他语言：http://example.com/en/api/v3/landingPage/json/54
